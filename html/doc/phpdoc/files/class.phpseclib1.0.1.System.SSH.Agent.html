Agent.php

Pure-PHP ssh-agent client.

PHP versions 4 and 5

Here are some examples of how to use this library: <?php include 'System/SSH/Agent.php'; include 'Net/SSH2.php';

$agent = new System_SSH_Agent();

$ssh = new Net_SSH2('www.domain.tld'); if (!$ssh->login('username', $agent)) { exit('Login Failed'); }

echo $ssh->exec('pwd'); echo $ssh->exec('ls -la'); ?>
